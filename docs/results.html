Executive Summary

This project produced a two-layer final result. The core strategy is a tuned QQQ channel breakout system with explicit volatility and risk-control rules: a 20-day breakout entry, a 20-day trend filter, a 0.20 ATR breakout-strength filter, a 2.0 ATR stop-loss, a 3.0 ATR trailing stop, a 15-day time stop, and a delayed breakout-failure exit. In the final out-of-sample evaluation window, this baseline strategy generated a Sharpe ratio of 1.34, a total return of 2.91%, a win rate of 81.8%, and a max drawdown of -0.81%.

On top of that baseline, the project added a supervised-learning overlay rather than a fully separate trading strategy. The final ML layer is a QQQ-only logistic model that uses a compact set of breakout-quality and implied-volatility features, including IV/HV spread, IV/HV ratio, and IV trend, to scale position size up or down. Under the final aggr4 scaling rule, the overlay improved the out-of-sample Sharpe ratio to 1.38 and total return to 3.40%, while slightly increasing max drawdown to -0.94%. Overall, the evidence suggests that the ML overlay is a small but real enhancement to an already viable breakout baseline, rather than a wholesale replacement for the strategy.

Baseline Summary

The final baseline is a tuned channel breakout strategy on QQQ with:

  • 20-day channel breakout entry
  • 20-day trend filter
  • 0.20 ATR breakout-strength filter
  • 2.0 ATR initial stop
  • 3.0 ATR trailing stop
  • 15-day timeout
  • 4-day breakout-failure delay

This baseline is already positive after costs and serves as the control strategy for the ML extension.

ML Overlay Summary

The final ML extension is not a fully separate strategy. It is a position-scaling overlay on top of the baseline breakout rule.

  • Asset scope: QQQ only
  • Model type: logistic classifier
  • Label framework: label then predict
  • Feature set: compact core_iv breakout and implied-volatility features
  • Use in backtest: scale position size mildly up or down rather than deleting all low-score trades

The resulting out-of-sample comparison shows a clear but still incremental improvement in both total return and Sharpe ratio relative to the baseline. The improvement comes from mild position scaling rather than a wholesale replacement of the breakout rule.

What The Final Result Means

The project now has two meaningful layers:

  1. A tuned baseline breakout strategy that is already tradable and positive after costs.
  2. A compact implied-volatility-aware ML overlay that improves out-of-sample Sharpe and total return on the showcase asset, while accepting a slightly deeper drawdown.

The improvement is not huge, and that is important to state honestly. This is not a magical black-box replacement for the breakout strategy. It is a small, interpretable enhancement on top of an already working baseline.
